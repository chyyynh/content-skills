# Hook 公式庫

clip 單獨使用（沒有 writer 腳本）時，根據內容類型選擇公式，填入模板生成 Hook 配音。

## Hook 配音長度

2-4 句，30-80 字，3-8 秒。不要只寫一句。

| 太短 | 適當 |
|------|------|
| "NASA 要送人上月球了" | "NASA 這週就要送人上月球了。四個太空人，十天的旅程，比人類歷史上任何人都更遠離地球。" |
| "AI 又有大新聞" | "OpenAI 昨天偷偷發了一個東西，整個矽谷都炸了。但 99% 的人還不知道這意味著什麼。" |
| "Cursor 有個大問題" | "如果你正在用 Cursor 寫程式碼，先停下來。我發現一個大多數人都忽略的問題，可能會讓你踩大坑。" |

## 公式模板

### 1. 懸念 — "你知道 _____ 嗎？"
- `你知道 [意外事實] 嗎？`
- `[主體] 有一個秘密，99% 的人不知道`

### 2. 顛覆 — "你以為的 _____ 全錯了"
- `你以為 [常見認知]？完全搞反了`
- `關於 [話題]，所有人都搞錯了一件事`

### 3. 緊急 — "在你 _____ 之前，先看這個"
- `在你 [常見行為] 之前，先看這個`
- `如果你正在 [行為]，立刻停下來`

### 4. 數據 — "只有 X% 的人知道"
- `只有 [小數字]% 的人知道 [有價值的事]`
- `[主體] 靠 [方法]，[時間] 內達到 [成果]`

### 5. 錯誤 — "X 個常見錯誤"
- `[動作] 的人都在犯的 [數字] 個錯`
- `別再 [錯誤做法] 了，試試這個`

### 6. 轉變 — "從 X 到 Y"
- `從 [差狀態] 到 [好狀態]，只用了 [方法/時間]`
- `[方法] 讓我的 [指標] 翻了 [倍數]`

### 7. FOMO — "你聽說了嗎？"
- `[新事物] 你聽說了嗎？`
- `[行業] 的人都在偷偷用 [工具/方法]`

### 8. 直球 — "X 分鐘搞懂"
- `[時間] 搞懂 [複雜概念]`
- `[話題] 最完整的懶人包`

## 選擇指南

| 內容類型 | 推薦 | 備選 |
|---------|------|------|
| 突發新聞/產品發布 | FOMO、緊急 | 懸念 |
| 爭議性觀點 | 顛覆 | 錯誤 |
| 工具評測/教學 | 緊急、錯誤 | 直球 |
| 數據報告/趨勢 | 數據 | 懸念 |
| 個人經驗/方法論 | 轉變 | 錯誤 |
| 科普解釋 | 直球 | 懸念、數據 |
