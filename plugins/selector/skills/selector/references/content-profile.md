# 內容定位

## 核心領域

- AI Agent × 金融：用 AI 代理幫助一般用戶做交易、監控市場、管理資產
- 2C 金融工具：面向散戶的智能投資、資訊聚合、決策輔助產品
- AI 基礎設施：讓上述應用成為可能的底層能力（MCP、tool use、多代理協作等）

## 關注的子方向

- AI agent 在交易/投資場景的應用（自動監控、訊號提醒、持倉分析）
- 金融資訊聚合與智能摘要（類似用 agent 起一個 OpenClaw 監控市場）
- 個人財務 AI 助手（記帳、稅務、理財規劃）
- Agent 框架與協議的演進（對金融場景有直接影響的）
- 相關融資與產品發布

## 不做的方向

- 純加密貨幣 / DeFi / Web3（除非涉及 AI agent 應用）
- 純技術教學（框架比較、coding tutorial）
- 純硬體（晶片、手機、消費電子）
- 遊戲、娛樂產業
- 與金融或 agent 無關的純 AI 研究論文

## 受眾畫像

- 對 AI 有基本認知的投資者、交易者
- 想用 AI 工具提升投資效率的散戶
- 關注 AI + 金融賽道的創業者和產品經理
- 核心問題：「這個 AI 能力怎麼幫我賺錢 / 省時間 / 降風險？」
